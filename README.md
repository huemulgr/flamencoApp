# flamencoApp
## Descripción
Este repositorio contiene el código fuente de aplicación android del proyecto Flamenco. 

## Requerimientos
* Sistema de login por Gmail
* Conexión al sistema de nube para obtener información de los MAS 

## Tecnologia
Android Studio 3.1.3
